친속관계 공증(길림지역)

아래 필요서류를 확인하신 후, 각 항목별로 파일 또는 사진을 업로드해 주세요. 주소는 반드시 정확하게 기재해 주셔야 하며, 추가 신청인이 있는 경우 해당 서류도 함께 첨부해 주세요.

필요서류 입력 및 업로드
1. 신청인 성명
2. 신청인 연락처
3. 신청인 주소 (정확히 기재하여 주세요)
4. 신청인 중국 신분증 앞면
5. 신청인 중국 신분증 뒷면
6. 신청인 여권 앞면
7. 신청인 중국 호구부 전체 페이지 (1)
8. 신청인 중국 호구부 전체 페이지 (2)
9. 추가 신청인 신분증 앞·뒷면 (1)
10. 추가 신청인 신분증 앞·뒷면 (2)
11. 추가 신청인 신분증 앞면 (1)
12. 추가 신청인 신분증 뒷면 (1)
13. 추가 신청인 신분증 앞면 (2)
14. 추가 신청인 신분증 뒷면 (2)
15. 추가 신청인 여권 앞면 (추가 신청 인원 여권 앞면을 모두 여기에 첨부)
16. 추가 신청인 호구부 전체 페이지 (1)
17. 추가 신청인 호구부 전체 페이지 (2)
18. 추가 신청인 호구부 전체 페이지 (3)
19. 기타 추가 서류
※ 업로드 가능한 형식: JPG, PNG, PDF
※ 여러 장의 서류가 있는 경우 하나의 PDF로 정리하거나, 여러 파일 업로드가 가능한 항목에 함께 첨부해 주세요.
※ 주소, 성명, 연락처는 정확히 입력해 주세요.
